HOW IT WORKS

1. SOMEONE CONNECTS TO WiFi
   └─ MikroTik captures: MAC + IP
   └─ Your system: Logs to database

2. BILLING PAGE LOADS
   └─ User selects package (e.g., "1 Hour - 15 KES")
   └─ User pays via Paystack

3. SYSTEM TRACKS DEVICE
   └─ POST /api/device-connection
   └─ Records: MAC, IP, device name, voucher
   └─ Status: CONNECTED

4. YOU CAN SEE THEM LIVE
   └─ GET /api/device-connection?action=active
   └─ Shows: X devices online right now

5. SESSION RUNS
   └─ User browses internet for purchased time
   └─ All tracked automatically

6. TIME EXPIRES
   └─ MikroTik auto-disconnects device
   └─ PUT /api/device-connection
   └─ Duration calculated and logged
